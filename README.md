# Bank Al-Andalous

Bank Al-Andalous is a smart digital banking web application built with Laravel, React, Inertia.js, and Tailwind CSS.

## Project Idea

Bank Al-Andalous aims to make banking easier, faster, and safer by offering users a modern digital platform where they can manage their money, track transactions, make payments, save automatically, and get AI-powered financial help.

## Main Features

- Secure authentication
- User dashboard with balance and transaction history
- Admin dashboard for managing users and accounts
- Deposits and withdrawals
- Money transfers
- Bill payments
- Auto monthly payments
- Digital Daret / group savings
- Saving goals with automatic deduction
- Saving simulator
- AI financial assistant
- Customer support chat
- Digital credit request
- Currency converter
- Appointment system

## Technologies Used

- Laravel
- React
- Inertia.js
- Tailwind CSS
- Vite
- SQLite / MySQL

## Project Goal

The goal of this project is to present a complete smart banking experience with a professional interface, useful digital services, and strong admin control.
